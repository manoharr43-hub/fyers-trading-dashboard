# 📈 Fyers Trading Dashboard

A Streamlit-based trading dashboard integrated with **Fyers API** and **NSE Stock Scanner**.

## 🚀 Features
- 🔐 Secure Fyers login and token generation  
- 💰 View profile, funds, holdings, and positions  
- 📊 NSE stock scanner with SMA20, SMA50, and RSI signals  
- ⬇️ Export scanner results to CSV  

## 🧩 Tech Stack
- Python 3.10  
- Streamlit  
- yfinance  
- fyers-apiv3  
- pandas
